# Image to PDF Converter

A modern React application to convert multiple images into a single PDF file.

## Features

- **Drag & Drop**: Easily upload multiple images.
- **Reorder**: Drag images to rearrange the page order.
- **PDF Generation**: Generate a PDF with one image per page.
- **Bilingual**: Supports English and Spanish.
- **Premium UI**: Built with TailwindCSS for a polished look.

## Running the Project

The application is currently running on port 5173.

```bash
http://localhost:5173/
```

## Development

To restart the server manually:

```bash
npm run dev
```

## Build

To build for production:

```bash
npm run build
```
